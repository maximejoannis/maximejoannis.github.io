Tableau des cas de test - Faire une réservation

ID Titre du test Préconditions Étapes Résultat attendu Statut Postcondition
CT-001 Demande de réservation par un Voyageur connecté L'utilisateur est connecté en tant que Voyageur 1. Consulter une annonce
2. Renseigner les informations requises
3. Soumettre la demande		 La réservation est créée et un message est envoyé à l’hôte À tester La réservation apparaît dans le tableau de bord de l'hôte
CT-002 Tentative de réservation par un utilisateur non connecté L'utilisateur est déconnecté 1. Consulter une annonce
2. Renseigner les informations requises
3. Soumettre la demande		 Le système empêche la soumission et demande à l’utilisateur de se connecter À tester L'utilisateur doit se connecter pour continuer
CT-003 Sélection de dates valides Voyageur connecté, logement disponible 1. Sélectionner une date de début valide
2. Sélectionner une date de fin valide postérieure
3. Soumettre la demande		 Réservation confirmée À tester La réservation est créée et bloquée pour d'autres utilisateurs
CT-004 Date de début dans le passé Voyageur connecté 1. Sélectionner une date de début dans le passé
2. Soumettre la demande		 Message d’erreur, la date de début doit être future À tester L'utilisateur doit sélectionner une nouvelle date
CT-005 Date de fin avant la date de début Voyageur connecté 1. Sélectionner une date de début valide
2. Sélectionner une date de fin antérieure
3. Soumettre la demande		 Message d’erreur, la date de fin doit être postérieure À tester L'utilisateur doit corriger la date de fin
CT-006 Sélection d'une période déjà réservée Voyageur connecté 1. Sélectionner une période déjà réservée
2. Soumettre la demande		 Message d’erreur, la période est indisponible À tester L'utilisateur doit choisir une autre période
CT-007 Saisie d'un nombre valide de voyageurs Voyageur connecté 1. Sélectionner des dates valides
2. Entrer un nombre de voyageurs > 0
3. Soumettre la demande		 Réservation confirmée À tester La réservation inclut le nombre de voyageurs spécifié
CT-008 Nombre de voyageurs nul Voyageur connecté 1. Sélectionner des dates valides
2. Saisir '0' voyageurs
3. Soumettre la demande		 Message d’erreur, le nombre doit être > 0 À tester L'utilisateur doit corriger l'entrée
CT-009 Nombre de voyageurs négatif Voyageur connecté 1. Sélectionner des dates valides
2. Saisir un nombre négatif
3. Soumettre la demande		 Message d’erreur, le nombre doit être positif À tester L'utilisateur doit corriger l'entrée
CT-010 Demande de réservation avec message Voyageur connecté 1. Sélectionner des dates valides
2. Saisir un nombre de voyageurs valide
3. Rédiger un message pour l’hôte
4. Soumettre la demande		 Réservation créée avec message À tester L'hôte reçoit le message avec la réservation
CT-011 Demande de réservation sans message Voyageur connecté 1. Sélectionner des dates valides
2. Saisir un nombre de voyageurs valide
3. Ne pas renseigner de message
4. Soumettre la demande		 Réservation créée sans message obligatoire À tester La réservation est créée sans message
CT-012 Réception d'une notification de réservation par l’hôte Voyageur connecté, soumission réussie 1. L’hôte consulte sa boîte de réception 'Messages'
2. L’hôte consulte sa boîte 'Réservations'		 Un nouveau message et une nouvelle réservation apparaissent À tester L'hôte voit la réservation et peut y répondre
CT-013 Accès aux réservations via le menu latéral Hôte connecté 1. Ouvrir le menu latéral
2. Accéder à la section 'Réservations'		 L’hôte voit la réservation récente À tester L'hôte peut gérer ses réservations
CT-014 Accès aux messages via le menu utilisateur Hôte connecté 1. Survoler le nom d’utilisateur
2. Sélectionner 'Messages'		 L’hôte voit le message de réservation À tester L'hôte peut répondre au message
