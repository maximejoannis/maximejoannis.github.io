# Selenium IDE – E2E HOMEY

Ce dossier contient les enregistrements Selenium IDE des parcours E2E de l’application HOMEY (gestion de locations saisonnières).

## Parcours couverts

- Voyageur – Contacter hôte
- Voyageur – Demande de réservation (non connecté)
- Voyageur – Demande de réservation (connecté)
- Hôte – Confirmation de demande de réservation
